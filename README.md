# sprint5-dashboard

## Dashboard de Anúncios de Carros nos EUA

Aplicativo web interativo criado com Streamlit para visualizar dados de anúncios de carros nos EUA.

## Funcionalidades

- Histograma de quilometragem dos carros
- Gráfico de dispersão de preço vs quilometragem
- Botões interativos para gerar os gráficos

## Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- Plotly Express
